# Screen Time and Sleep Relationship Analysis

## Project Overview
This project explores the relationship between daily screen time and sleep patterns. The goal is to investigate whether higher screen time is associated with shorter sleep duration or lower sleep quality.

## Motivation
Screen time has become a major part of daily life, especially with smartphones and digital devices. Understanding its relationship with sleep can provide useful behavioral insights and help explain patterns in daily well-being.

## Research Question
Does increased daily screen time negatively affect sleep duration and sleep quality?

## Planned Data
The dataset will include:
- Daily screen time records
- Sleep duration data
- Sleep quality indicators
- Possible additional variables such as weekday/weekend information

## Planned Methods
- Data collection and cleaning
- Exploratory data analysis
- Hypothesis testing
- Regression and/or classification models
- Data visualization

## Expected Outcome
The project aims to identify whether there is a meaningful relationship between screen time and sleep, and to build a simple predictive framework using behavioral data.

## Repository Structure
- `data/` for raw and processed data
- `notebooks/` for analysis notebooks
- `src/` for Python scripts
- `reports/` for proposal and final report
